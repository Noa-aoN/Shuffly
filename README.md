# Shuffly

## アプリを作った理由
- どうしてこのアプリを作ろうと思ったのか（未編集）

## できること（機能一覧）
- メンバーの手軽な入力 / インポート
- グループの自動割り当て（履歴を保持）
- 順番の自動割り当て
- 役割の自動割り当て
- 結果のコピー・エクスポート

## 技術スタック
- Ruby（3.2.3）
- Ruby on Rails（7.2.3）
- Docker / docker-compose
- PostgreSQL
